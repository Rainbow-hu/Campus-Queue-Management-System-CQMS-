Campus Queue Management System (CQMS)

Project Description
The Campus Queue Management System is a simple Python-based application designed to manage student queues in campus offices such as bursary, ICT unit, or registry. It helps ensure students are attended to in the order they arrive.

Features
- Add students to a queue
- Serve students in FIFO order
- Display the current queue
- Prevents serving from an empty queue

Technologies Used
- Python 3
- Object-Oriented Programming (OOP)

How to Run the Project
1. Ensure Python is installed on your system
2. Download or clone the repository
3. Open a terminal in the project folder
4. Run the command:

   python campus_queue_system.py
